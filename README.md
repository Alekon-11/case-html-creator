# Генератор HTML-страниц (кейс)

Веб-инструмент для сборки одностраничного кейса по фиксированному шаблону (свёрстан по макету Figma). Вы заполняете данные по блокам, настраиваете цвета и картинки — на выходе получаете **один самодостаточный `.html`-файл** со встроенными стилями, скриптами и изображениями (base64). Без фреймворков и сборки — чистые HTML, CSS, JS.

## Возможности

### Интерфейс
- **Двухпанельный редактор:** слева — настройки по блокам, справа — живое превью в `iframe`.
- **Превью в реальном времени** с переключением ширины: Десктоп / Планшет (768px) / Телефон (390px).
- **Скачивание** результата кнопкой — один файл `case-page.html`.
- **Экспорт в PDF** кнопкой «Скачать PDF» (печать средствами браузера → «Сохранить как PDF»; векторный PDF с выделяемым текстом, print-стили включены).

### Блоки
Каждый блок включается/выключается тумблером **«Добавить блок на страницу»** — выключенный блок не попадает в результат. Доступные блоки:

| Блок | Что настраивается |
|---|---|
| **Hero (обложка)** | фоновое изображение, логотип клиента, тег, заголовок, подзаголовок, метрики (значение + подпись). Фон по умолчанию — Акцент 2 |
| **Клиент** | заголовок, текст |
| **Цели** | заголовок, список пунктов, цвет маркера |
| **Задачи кампании** | заголовок, список пунктов, цвет маркера |
| **Креативы** | заголовок, любое число изображений (по умолчанию 1) |
| **Инструменты** | заголовок, теги-чипсы, параметры (Период/Гео и т.п.) |
| **Каналы / Форматы** | две колонки тегов, примечание с настраиваемым цветом текста |
| **Целевая аудитория** | заголовок, текст, теги-сегменты |
| **Стратегия** | заголовок, абзацы, список ключевых шагов |
| **Результаты** | абзацы, список (после текста), карточки метрик, доп. блок «Баннеры», блок-отзыв (текст, автор, должность, фото) |
| **Форма** | заголовок, подзаголовок, текст кнопки |
| **Кейсы** | карточки-слайдер (см. ниже) |

### Цвета
- **Палитра из 5 цветов:** 2 контрастных (текст, фон) + 3 акцентных. Применяются ко всей странице через CSS-переменные.
- **Точечные цвета:** маркеры списков (Цели / Задачи / Результаты), цвет текста примечания, цвет кейса и цвет текста тега кейса.

### Изображения
- Загрузка через drag&drop или клик.
- Встраиваются в итоговый файл как **base64** — страница остаётся самодостаточной (один файл, без внешних картинок).

### Текст
- В любом текстовом поле можно сделать фрагмент **жирным** разметкой `**вот так**`.
- Переносы строк сохраняются.

### Блок «Кейсы» — слайдер
- Показывает 2 кейса на десктопе; при бóльшем числе появляются **стрелки пролистывания** (SVG).
- На мобильных виден 1 кейс, стрелки появляются автоматически (по факту переполнения).
- У каждого кейса: изображение, тег, заголовок, до 3 метрик (значение + подпись), кнопка-**ссылка** с настраиваемым URL.
- Цвет кейса задаёт цвет метрик и фон тега; цвет текста тега настраивается отдельно.

### Анимация
- Блоки **плавно появляются при скролле** (`IntersectionObserver`, fade + сдвиг). Логика встроена в итоговый файл.

### Технологии
- Ваниль: **HTML + CSS + JS**, без фреймворков и зависимостей.
- Шрифт макета — **Gilroy** (платный); в итоговом файле используется он при наличии в системе, иначе бесплатный фолбэк **Manrope** (Google Fonts).

## Структура проекта

```
public_html/
├── index.html    # каркас редактора (две панели)
├── styles.css    # стили интерфейса редактора
├── app.js        # схема блоков, форма, состояние, генерация HTML, превью, экспорт
└── README.md
```

> Важно: `index.html`, `styles.css`, `app.js` — это сам **редактор**. Итоговая страница-кейс собирается в один HTML-файл целиком внутри `app.js` (функция `generateHTML`) и скачивается кнопкой.

## Запуск локально

Это статический сайт — сборка не нужна.

- Открыть `index.html` напрямую в браузере, **или**
- Поднять любой статический сервер, например:
  ```bash
  npx serve .
  # или
  python -m http.server 8000
  ```

## Как пользоваться

1. Откройте редактор.
2. Настройте палитру (2 контрастных + 3 акцентных цвета).
3. Заполните нужные блоки; ненужные — выключите тумблером.
4. Загрузите изображения (Hero, Креативы, Кейсы, фото автора отзыва).
5. Проверьте результат в превью (в т.ч. на ширине «Телефон»).
6. Нажмите **«Скачать HTML»** — получите готовый `case-page.html`, либо **«Скачать PDF»** — откроется диалог печати → «Сохранить как PDF».

## Деплой

Статический сайт. На Vercel определяется автоматически (Framework Preset: **Other**, без build-команды), корень — папка с `index.html`.
